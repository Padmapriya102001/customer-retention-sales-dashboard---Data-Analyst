# Customer Retention & Sales Performance Dashboard

## Project Overview

This project analyzes customer behavior, sales trends, product performance, and regional performance using SQL, Python, and Power BI.

## Tools Used

- SQL
- Python
- Power BI
- Excel

## Key KPIs

- Total Sales
- Total Profit
- Total Customers
- Total Orders

## Dashboard Features

- Monthly Sales Trend
- Product Performance Analysis
- Regional Performance Analysis
- Customer Insights
- Interactive Filters

## Business Insights

- Identified top-performing products and regions.
- Analyzed customer purchasing behavior.
- Tracked sales and profit trends over time.

## Project Workflow

1. Data Collection
2. Data Cleaning using Python
3. SQL Analysis
4. Dashboard Creation in Power BI
5. Business Insights Generation
